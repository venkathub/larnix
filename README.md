# Larnix

A free, open-access, beginner-to-frontier school for AI/ML — modeled on Zerodha Varsity's pedagogy (plain language, analogies, short single-idea chapters, "Key Takeaways", difficulty tiers, quizzes, certification), where every chapter ships runnable code and a complete beginner on a low-spec laptop can become job-ready.

See [`docs/ROADMAP.md`](docs/ROADMAP.md) for the phased build plan (P0–P7).

## Try it (5 minutes)

The Phase P0 platform is proven on one zero-install sample chapter — *Train your
first model, in your browser*. Follow [`docs/WALKTHROUGH.md`](docs/WALKTHROUGH.md)
to click and run it locally or on the deployed site. Build/preview and CI details
are in [`docs/RUNBOOK.md`](docs/RUNBOOK.md); the design rationale lives in
[`docs/DECISIONS.md`](docs/DECISIONS.md).
