A 10-minute history of AI

Beginner Browser Stable

The hook

Generative AI feels like it arrived overnight — ChatGPT reached the public in late 2022 and the world changed its mind about computers in a year. But the first artificial neuron was described in 1943, before the transistor existed. AI is roughly eighty years old. Its story is not a straight climb; it is a series of bold promises, crushing disappointments (“AI winters”), and a handful of breakthroughs that changed everything.

Here is the ten-minute version.

You’ll need from before: the family tree from Chapter 4 — AI ⊃ ML ⊃ deep learning ⊃ generative AI. We’ll watch those rings appear, in order, over time.

The story, in five acts

Act 1 — The idea (1943–1958). In 1943, Warren McCulloch and Walter Pitts wrote down a simple mathematical model of a neuron. In 1950, Alan Turing asked “can machines think?” and proposed a test for it. In 1956, a summer workshop at Dartmouth College gave the field its name: artificial intelligence. Then, in 1958, Frank Rosenblatt built the Perceptron — a machine that learned to tell patterns apart from examples. Optimism ran high.

Act 2 — The first winter (1969–1980s). In 1969, Marvin Minsky and Seymour Papert proved that a single-layer perceptron could not learn even simple patterns (famously, the “exclusive-or”). The limitation was real, the disappointment sharp, and funding collapsed. This was the first AI winter — a recurring pattern where hype outruns results and the money leaves.

Act 3 — Learning returns (1986–2010). In 1986, a method called backpropagation (popularised by Rumelhart, Hinton, and Williams in Nature) let networks with many layers learn — sidestepping the 1969 limitation. Neural networks revived; by the 1990s they could read handwritten digits. Meanwhile, a different, rule-and-search kind of AI hit a milestone: in 1997, IBM’s Deep Blue beat world chess champion Garry Kasparov. (Deep Blue searched moves by rules — it did not learn, a useful reminder that “AI” is broader than “machine learning.”)

Act 4 — The deep-learning boom (2012–2017). In 2012, a deep network called AlexNet (Krizhevsky, Sutskever, and Hinton) won the ImageNet image-recognition contest by a wide margin. Three things had finally come together: big datasets, fast GPUs, and deeper networks. Deep learning went from fringe to mainstream almost overnight. In 2016, DeepMind’s AlphaGo beat Go champion Lee Sedol — a game long thought too intuitive for computers.

Act 5 — Transformers and generative AI (2017–today). In 2017, eight researchers at Google published “Attention Is All You Need,” introducing the transformer — the architecture behind nearly every modern language model. Scaled up on enormous text, transformers became foundation models: trained once, then adapted to countless tasks. GPT-3 (2020) showed how capable they were; ChatGPT (2022) put one in everyone’s hands. The current frontier extends this with models trained to generate images and audio, and reasoning models trained to work through problems step by step.

🔬 Going deeper (optional): notice the rhythm — an idea, a wave of hype, a winter, then a breakthrough that makes the old promise real. Today’s excitement is the warmest summer yet, but the pattern is worth remembering.

Key Takeaways

Key Takeaways

  1. AI is about eighty years old — the artificial neuron dates to 1943.
  2. Progress came in booms and winters: hype, disappointment, then a breakthrough that delivered on the old promise.
  3. The 2012 deep-learning boom (AlexNet) and the 2017 transformer are the two milestones that most directly enabled today’s generative AI.
  4. “AI” is broader than “machine learning”: Deep Blue (1997) was AI that searched by rules, with no learning at all.

Exercise 1 — Guided (≈5 min) 🟢

Complete the marked line so event_year returns the year a named event happened (or None if it isn’t in the timeline). Replace ____, then press Run.

If you’re stuck The loop has already unpacked each entry into year and event. When event matches, which of those two should the function return?
Show solution 
timeline = [
    (1958, "Rosenblatt's Perceptron"),
    (2012, "AlexNet wins ImageNet"),
    (2017, "The transformer"),
]

def event_year(timeline, name):
    for year, event in timeline:
        if event == name:
            return year
    return None
When the event matches, return its year; if the loop finishes with no match, the event isn’t in the timeline, so return None.

Exercise 2 — Implement (≈10 min) 🟡

Write events_in_era(timeline, start, end) — return the list of event names whose year is between start and end, inclusive. Complete the marked line.

If you’re stuck “Between start and end, inclusive” is two comparisons joined with and: the year is at least start and at most end. Python lets you chain them: start <= year <= end.
Show solution 
timeline = [
    (1958, "Perceptron"),
    (1986, "Backpropagation"),
    (2012, "AlexNet"),
    (2017, "Transformer"),
    (2022, "ChatGPT"),
]

def events_in_era(timeline, start, end):
    names = []
    for year, event in timeline:
        if start <= year <= end:
            names.append(event)
    return names
start <= year <= end is True only for years inside the era, so only those event names are collected — in the order they appear in the timeline.
